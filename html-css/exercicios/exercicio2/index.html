Paragrafos e Quebras

Você pode escrever um paragrafo de qualquer jeito. Basta colocar tudo dentro de um par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar especifico
como esse, você pode usar a tag <br>

Usando emojis no site 🥰

Para incluir emojis em seu site, basta acessar o site emojipedia.org e copiar o código ou o próprio emoji e colar no editor de código onde deseja que o emoji apareça no texto.
Após escolher o emoji desejado na emojipedia, basta rolar a pagina até em baixo e pegar o código do emoji.
Atenção! Para usar o emoji através do código (o que é recomendo devido ao copiar e colar não funcioar em todos os navegadores). Deve substituir o U+ que aparece no inicio pelo E comercial + um simbolo de cerquilha + a letra x em minusculo. Por exemplo:
Saudação vulcana 🖖

Simbolos especiais

É possivel inserir alguns simbolos especiais através de códigos, os códigos são chamados de HMTL Entities. Segue alguns exemplos de simbolos que podem ser incluidos:

Marca registrada = ®
Copyright = ©
Trade Mark = ™
Moeda euro = €
Moeda Yen = ¥
Moeda libra = £
Centavo = ¢
Vazio = ∅
Soma = ∑
Delta = Δ
Seta Esquerda = ←
Seta Direta = →
Seta para cima = ↑
Seta para baixo = ↓
